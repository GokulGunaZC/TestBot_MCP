# React Vite Customer Success Console

This fixture is a public React + Vite + TypeScript customer success console. Users can review accounts, filter renewal risk, and submit a success plan.

Acceptance criteria:
- The home route shows Customer Success Console, Renewal Risk Queue, and Northwind Robotics.
- The accounts route lets users filter by High Risk and shows At Risk Accounts.
- The plan route has a Success Plan form with Account Name, Renewal Owner, and Save Plan.
- The app is public and must not require authentication.
