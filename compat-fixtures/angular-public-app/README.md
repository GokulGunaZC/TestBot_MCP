# Angular Public Project Portal

This fixture is a public Angular-style project portal. Users can review project health, filter delivery risks, submit a planning request, and navigate between dashboard, intake, and reports routes.

Acceptance criteria:
- The dashboard route shows Project Health Center, Active Delivery Streams, and Sprint Confidence.
- The intake route validates a Workstream Name before showing Intake Saved.
- The reports route shows Release Readiness and Export Report.
- All routes are public and do not require authentication.
