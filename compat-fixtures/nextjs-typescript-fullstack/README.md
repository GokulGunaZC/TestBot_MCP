# Next.js TypeScript Operations Hub

This fixture is a public Next.js TypeScript fullstack operations hub with pages and API route handlers.

Acceptance criteria:
- The home page shows Operations Hub, Incident Queue, and Payments Latency.
- The customers page shows Customer Portfolio and Summit Foods.
- The API exposes GET `/api/incidents`, POST `/api/incidents`, and GET `/api/metrics`.
- Invalid incident creation returns a validation error.
